# Interv-AI
AI-powered interview report generator that turns candidate interviews into structured, professional evaluation reports instantly.

Generate professional interview reports with AI
Structured scoring and candidate insights
Export-ready report formats
Fast, minimal, and easy to integrate

Tech Stack

Frontend – ReactJS
Backend – Node.js
AI – GEMINI API
Database – MONGODB

git clone https://github.com/shubham-m77/interv-ai.git
cd frontend: for frontend
npm install
npm run dev

cd backend: for backend
npm install
npm run dev
